# LittleHabits
这是一个习惯（目标）打卡类的安卓app，基本功能为：小目标编辑、目标打卡、时钟倒计时、图表分析、拍照设置头像、百度定位等。



### 运行

1、克隆项目源码到本地

2、使用Android Studio来打开该项目，并配置好安卓模拟器或连接安卓真机。

3、点击运行按钮即可运行。
